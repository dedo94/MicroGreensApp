# Contributing

This is a personal project I built for my own microgreens tracking and
maintain in my spare time. The repository is public for reference, but
I'm not accepting external pull requests or feature requests right now.

- **Bugs**: feel free to open an issue, I'll look at it when I can.
- **Pull requests**: unsolicited PRs will generally be closed unread —
  not because they aren't appreciated, but because this app is scoped
  tightly to how I personally use it. Open an issue first if you want to
  discuss a change.
- **Security issues**: see [SECURITY.md](SECURITY.md).
